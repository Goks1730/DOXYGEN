
# DOXYGEN

Documentation generator


## Deployment
STEP 1: Update Your Server

```bash
  sudo apt update
```
STEP 2: Install Doxygen

```bash
  sudo apt install doxygen
```
STEP 3: Check Version

```bash
  doxygen --version
```
